We have 3 folders here. 

folder phase 1 contains our initial EDA (Exploratory Data Analysis). 

folder phase 2 contains the dataset and the code which creates the final data (rules_conf.xlsx)

folder phase 3 contains the final phase work. To execute our phase3 please click on the index.html. 
This will open a webpage, where you can select the products and hit "BUY". On the next panel (which 
generates after clicking the BUY button) you will see product recommendations.
